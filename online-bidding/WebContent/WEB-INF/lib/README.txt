REQUIRED JAR FILES FOR AUCTION SYSTEM

Please download and place the following JAR files in this directory:

1. mysql-connector-java-8.0.28.jar
   - Download from: https://dev.mysql.com/downloads/connector/j/
   - Required for MySQL database connectivity

2. jstl-1.2.jar
   - Download from: https://mvnrepository.com/artifact/javax.servlet/jstl/1.2
   - Required for JSP Standard Tag Library

3. servlet-api.jar (if not provided by Tomcat)
   - Usually included with Tomcat installation
   - Required for Servlet API

DOWNLOAD LINKS:
- MySQL Connector: https://repo1.maven.org/maven2/mysql/mysql-connector-java/8.0.28/mysql-connector-java-8.0.28.jar
- JSTL: https://repo1.maven.org/maven2/javax/servlet/jstl/1.2/jstl-1.2.jar

After downloading, restart Tomcat for changes to take effect.
